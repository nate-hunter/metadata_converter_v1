XML to CSV Metadata File Converter

About:

XMLs aren't the most friendly data format to read. Scheduling a Library TV season of many episodes can be tedious - and especially annoying when on a deadline.

Metadata in a CSV format is a much more user friendly approach as the metadata is in one file and each cell can be copied and pasted.

This program converts selected XMLs into a single CSV file which can be exported, saved, and viewed as a table.
